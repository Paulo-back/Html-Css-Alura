## ✨ Destaques

- **Design responsivo mobile-first** — funciona bem de celular a desktop.
- **Identidade "editor de código"** — o menu é uma barra de abas (`index.html` / `sobre.html` /
  `curriculo.html`), como abas de arquivos abertos num editor.
- **Trilha de aprendizado no estilo `git log`** — a página "Sobre" apresenta minha formação
  complementar como um histórico de commits, com data, hash e link pro certificado.
- **Página de currículo estruturada** — resumo, experiência, formação e certificações
  organizados em cards, com botão para baixar o PDF.
- **Acessibilidade básica cuidada** — foco visível no teclado e respeito a
  `prefers-reduced-motion`.

## 🛠️ Tecnologias utilizadas

- **HTML5** — estruturação semântica do conteúdo.
- **CSS3** — variáveis (custom properties), Flexbox e CSS Grid para o layout.
- **Design responsivo** — breakpoints mobile-first.
- **SVG inline** — ícones (GitHub, LinkedIn, Instagram, contato) sem dependência de imagens
  externas.
- **Google Fonts** — Space Grotesk, Inter e JetBrains Mono.

## ▶️ Como rodar localmente

Não há build nem dependências — é só abrir o `index.html` no navegador, ou, se preferir um
servidor local:

```bash
# com Python
python3 -m http.server 8000

# ou com a extensão Live Server do VS Code
```
